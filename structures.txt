================================================================================
ПОЛНЫЙ СПИСОК СТРУКТУР ДАННЫХ В ПРОГРАММИРОВАНИИ
================================================================================

1. ПРИМИТИВНЫЕ ТИПЫ (основные строительные блоки)
--------------------------------------------------------------------------------
- Целые числа (Integer, Long, Short, Byte)
- Числа с плавающей точкой (Float, Double)
- Символы (Char)
- Логический тип (Boolean)
- Указатели / Ссылки (Pointer, Reference)
- Перечисления (Enum)
- Void / Unit (отсутствие значения)

2. ЛИНЕЙНЫЕ СТРУКТУРЫ (последовательное хранение)
--------------------------------------------------------------------------------
- Массив (Array) — статический, фиксированный размер                                +
- Динамический массив (ArrayList, Vector, List — динамическое расширение)           +
- Связный список (Linked List)                                                      +
    - Односвязный список (Singly Linked List)                                       +
    - Двусвязный список (Doubly Linked List)                                        +
    - Кольцевой (циклический) список (Circular Linked List)                         +
- Стек (Stack) — LIFO (Last In, First Out)                                          +
- Очередь (Queue) — FIFO (First In, First Out)                                      +
    - Простая очередь                                                               +
    - Кольцевая очередь (Circular Queue)                                            +
    - Дек (Deque — Double Ended Queue) — двусторонняя очередь                       +
    - Приоритетная очередь (Priority Queue)                                         +
- Разреженный массив (Sparse Array)                                                 -

3. АССОЦИАТИВНЫЕ СТРУКТУРЫ (ключ-значение)
--------------------------------------------------------------------------------
- Хеш-таблица (Hash Table)
- HashMap / Dictionary / Map
- Хеш-множество (HashSet)
- LinkedHashMap (сохраняет порядок вставки)
- TreeMap (отсортированная по ключам)
- Контейнер на основе открытой адресации (Open Addressing)
- Контейнер на основе цепочек (Chaining)
- Кастомная хеш-функция
- Фильтр Блума (Bloom Filter) — вероятностная структура

4. ДЕРЕВЬЯ (иерархические структуры)
--------------------------------------------------------------------------------
- Бинарное дерево (Binary Tree)
- Двоичное дерево поиска (BST — Binary Search Tree)
- Сбалансированные деревья:
    - AVL-дерево
    - Красно-чёрное дерево (Red-Black Tree)
    - Splay-дерево (косое)
    - Декартово дерево (Treap — Tree + Heap)
- B-деревья и их вариации:
    - B-дерево
    - B+ дерево
    - B* дерево
- Дерево отрезков (Segment Tree)
- Дерево Фенвика (Fenwick Tree / Binary Indexed Tree)
- Префиксное дерево (Trie / Radix Tree / Prefix Tree)
- Куча (Heap):
    - Двоичная куча (Binary Heap)
    - Фибоначчиева куча (Fibonacci Heap)
    - Биномиальная куча (Binomial Heap)
    - Min-Heap и Max-Heap
- Дерево интервалов (Interval Tree)
- Дерево диапазонов (Range Tree)
- КД-дерево (KD-Tree)
- Дерево квадрантов (Quadtree)
- Октри (Octree)
- Дерево ван Эмде Боаса (Van Emde Boas Tree)
- Дерево выражений (Expression Tree)
- Дерево решений (Decision Tree)

5. ГРАФЫ (сетевые структуры)
--------------------------------------------------------------------------------
- Ориентированный граф (Directed Graph)
- Неориентированный граф (Undirected Graph)
- Взвешенный граф (Weighted Graph)
- Разреженный граф (Sparse Graph)
- Плотный граф (Dense Graph)
- Гиперграф (Hypergraph)
- Двудольный граф (Bipartite Graph)
- Полный граф (Complete Graph)
- Планарный граф (Planar Graph)
- Способы представления:
    - Матрица смежности (Adjacency Matrix)
    - Список смежности (Adjacency List)
    - Массив рёбер (Edge List)
    - Матрица инцидентности (Incidence Matrix)

6. МНОЖЕСТВА (Set)
--------------------------------------------------------------------------------
- Битовое множество (BitSet / Bit Array)
- Отсортированное множество (SortedSet)
- Множество на основе хешей (HashSet)
- Мультимножество (Multiset / Bag — допускает дубликаты)

7. СПЕЦИАЛИЗИРОВАННЫЕ СТРУКТУРЫ
--------------------------------------------------------------------------------
- Запись (Record / Struct / Tuple)
- Объединение (Union / Variant)
- Disjoint Set Union (DSU / Union-Find) — система непересекающихся множеств
- Массив с пропусками (Sparse Set)
- Кэш (Cache):
    - LRU Cache (Least Recently Used)
    - LFU Cache (Least Frequently Used)
- Буфер (Buffer):
    - Кольцевой буфер (Ring Buffer / Circular Buffer)
- Базовый массив (Packed Array)
- Массив с выравниванием (Aligned Array)
- Persistent Data Structures (персистентные)
    - Персистентный стек
    - Персистентный массив
    - Персистентное дерево отрезков
- Immutable Data Structures (неизменяемые)
- Гамильтонов путь / граф (спец. структура)

8. СТРУКТУРЫ ДЛЯ ПОТОКОВ ДАННЫХ (Streaming)
--------------------------------------------------------------------------------
- Скользящее окно (Sliding Window)
- Сэмплер (Sampler — Reservoir Sampling)
- HyperLogLog (приблизительное подсчитывание уникальных элементов)
- Count-Min Sketch
- T-Digest (квантильная оценка)

9. СТРОКИ КАК СПЕЦИФИЧЕСКАЯ СТРУКТУРА
--------------------------------------------------------------------------------
- C-строка (null-terminated)
- Pascal-строка (с длиной)
- Rope (верёвка — для эффективной конкатенации)
- StringBuilder / StringBuffer
- Шаблоны строк (String templates)
- Интернированные строки (Interned Strings)

10. ГИБРИДНЫЕ И ПРОДВИНУТЫЕ
--------------------------------------------------------------------------------
- Список с пропусками (Skip List)
- Матрица (Matrix) — многомерный массив
- Тензор (Tensor) — многомерный аналог матрицы
- Хеш-массив (Hash Array Mapped Trie — HAMT)
- Дерево с указателями на родителя (Parent Pointer Tree)
- Приоритетный дек (Priority Deque)
- Y-fast trie
- X-fast trie
- Dancing Links (для точного покрытия)

11. СТРУКТУРЫ ИЗ ФУНКЦИОНАЛЬНОГО ПРОГРАММИРОВАНИЯ
--------------------------------------------------------------------------------
- Список (Cons-list / Lisp list)
- Ленивый список (Lazy List / Stream)
- Монада (Maybe, Either, List Monad)
- Zipper (для навигации по дереву)
- Конечные автоматы (Finite State Machine)

12. СИСТЕМНЫЕ / НИЗКОУРОВНЕВЫЕ
--------------------------------------------------------------------------------
- Битовая карта (Bitmap)
- Битовая маска (Bitmask)
- Буфер фиксированного размера (Fixed-size buffer)
- Page / Block (в системах БД и файловых системах)
- Регистровый файл (Register file — CPU)

13. БАЗЫ ДАННЫХ И ИНДЕКСЫ (как структуры данных)
--------------------------------------------------------------------------------
- Индекс с плотной структурой (Dense Index)
- Индекс с разреженной структурой (Sparse Index)
- Перевёрнутый индекс (Inverted Index)
- Пространственный индекс (Spatial Index — R-tree, R+tree)
- Битовый индекс (Bitmap Index)
- Хеш-индекс (Hash Index)

14. СТРУКТУРЫ ДАННЫХ ДЛЯ ГЕОМЕТРИИ
--------------------------------------------------------------------------------
- Сетка (Grid)
- Полигональная сетка (Mesh)
- BSP-дерево (Binary Space Partitioning)
- Выпуклая оболочка (Convex Hull — структура)
- Полуплоскость (Half-space representation)

15. ПРОЧИЕ / ЭКЗОТИЧЕСКИЕ
--------------------------------------------------------------------------------
- XOR-связный список (XOR Linked List — экономит память)
- Стек с одним указателем (One-pointer stack)
- Котёл (Beaker — структура из синтеза очереди и стека)
- Куча с поддержкой удаления (Deletable Heap)
- Multiset с порядком (Order Statistic Tree)
- Распределённые структуры (DHT — Distributed Hash Table, Chord, Kadmelia)
